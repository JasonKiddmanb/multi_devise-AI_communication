# V1 安装前检查清单

> 在开始之前，逐项确认你的环境就绪。

## 硬件要求

- [ ] **PC 一台**（Windows 10/11）
  - 建议 16GB+ 内存
  - 如有 NVIDIA 显卡体验更好（可选）
- [ ] **iPhone**（iOS 15+）

## 账号准备

- [ ] **Tailscale 账号** — 用 Google/Microsoft/Apple 账号即可，免费版够用
- [ ] 同一个 Tailscale 账号安装在 PC 和 iPhone 上

## 安装清单

- [ ] **Tailscale（PC）** — https://tailscale.com/download
- [ ] **Tailscale（iPhone）** — App Store
- [ ] **Ollama（PC）** — https://ollama.com/download
- [ ] **Docker Desktop（PC）** — https://docs.docker.com/desktop/setup/install/windows-install/

## 网络检查

- [ ] PC 和 iPhone 都能上网
- [ ] PC 防火墙没有阻止 Tailscale（UDP 41641）
- [ ] 公司/VPN 网络可能限制 Tailscale 流量

## 如果遇到问题

逐项确认后仍有问题，请告诉我具体的报错信息。
